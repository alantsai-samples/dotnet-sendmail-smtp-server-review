# dotnet-sendmail-smtp-server-review

這個專案是用來提供測試寄送mail的demo程式碼。

## 快速使用 (getting started)

把程式碼clone下來或者下載下來，用Visual Studio打開，修改`App.config`測試不同寄送mail的方式。
